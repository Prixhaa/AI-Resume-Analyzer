## 📌 Project Overview

AI Resume Analyzer is a machine learning and NLP project designed to automate resume screening. The system extracts information from PDF resumes, identifies technical skills, and serves as the foundation for an Applicant Tracking System (ATS).

This project is being developed as part of my AI Engineering portfolio and demonstrates practical applications of Python, Natural Language Processing, and resume analytics.
